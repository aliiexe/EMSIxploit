# News media

Drop images (or other media) for news feed items here.

- Use descriptive filenames, e.g. `ctf-workshop-march.jpg`, `meetup-q1.png`.
- Reference them in `data/news.ts` with the path `/news/filename.ext` (e.g. `image: "/news/ctf-workshop-march.jpg"`).

Supported formats: jpg, jpeg, png, gif, webp.
