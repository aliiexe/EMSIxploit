# Team member photos

Place each team member's photo in this folder.

**Naming:** Use lowercase, hyphenated names matching the `avatar` value in `data/members.ts`, for example:

- `alex-chen.jpg` for Alex Chen
- `jordan-lee.png` for Jordan Lee

**Recommended:** Square or near-square images (e.g. 400×400px or 600×600px). Format: JPG or PNG.

After adding a file, set the member's `avatar` in `data/members.ts` to `"/team/filename.jpg"` (or the filename you used).
