# About page media slider

Drop **images** (jpg, png, webp) and **videos** (mp4, webm) here for the automatic slider on the about page.

Then add their filenames to `data/team-slider.ts` (e.g. `"/team-slider/your-photo.jpg"` or `"/team-slider/your-video.mp4"`).

Videos will play in loop, muted. The slider scrolls automatically with a fade-to-transparent effect on the edges.
